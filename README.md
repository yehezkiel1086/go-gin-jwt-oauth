# Go Gin JWT OAuth2

A template of Go JWT and OAuth2 implementation built with Gin framework and Hexagonal architecture.

## Requirements

1. Go >= 1.24.4
2. Docker >= 28.3.2
3. GNU Make >= 4.4.1

## Installation

1. Install go libraries: `go mod tidy`
2. Compose up: `docker compose up -d`
3. Run: `go run cmd/main.go`
